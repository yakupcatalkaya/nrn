56 CVODE expects dy/dt = f(y) and solve (I - gamma*J)*x = b with approx to
57 J=df/dy.
58 
59 The NEURON fixed step method sets up C*dv/dt = F(v)
60 by first calculating F(v) and storing it on the right hand side of
61 the matrix equation ( see src/nrnoc/treeset.c nrn_rhs() ).
62 It then sets up the left hand side of the matrix equation using
63 nrn_set_cj(1./dt); setup1_tree_matrix(); setup2_tree_matrix();
64 to form
65 (C/dt - J(F))*dv = F(v)
66 After a nrn_solve() the answer, dv, is stored in the right hand side
67 vector.
68 
69 However, one must be aware of the fact that the cvode state vector
70 y is not the vector y for the fixed step in two ways. 1) the cvode
71 state vector includes ALL states, including channel states.
72 2) the cvode state vector does NOT include the zero area nodes
73 (since the capacitance for those nodes are 0). Furthermore, cvode
74 cannot work with the extracellular mechanism (both because extracellular
75 capacitance is often 0 and because more than one dv/dt is involved
76 in some of the current balance equations) or LinearMechanism (same reasons).
77 In that case the current balance equations are of the differential
78 algebraic form c*dv/dt = f(v) where c is non-diagonal and may have empty rows.
79 The variable step method for these cases is handled by daspk.

775 v at nodes with capacitance is correct (from scatter v) however
776 v at no-cap nodes is out of date since the values are from the
777 previous call. v would merely be the weighted average of
778 the adjacent v's except for the possibility of membrane
779 currents at branch points. We thus need to calculate both i(v)
780 and di/dv at those zero area nodes so that we can solve the
781 algebraic equation (di/dv + a_j)*vmnew = - i(vmold) + a_j*v_j.
782 The simplest case is no membrane current and root or leaf. In that
783 case vmnew = v_j. The next simplest case is no membrane current.
784 In that case, vm is the weighted sum (via the axial coefficients)
785 of v_j.

794 
795  for (i = 0; i < z.no_cap_count_; ++i) { // initialize storage
796  Node* nd = z.no_cap_node_[i];
797  NODED(nd) = 0;
798  NODERHS(nd) = 0;
799  }
800  // compute the i(vmold) and di/dv
801  rhs_memb(z.no_cap_memb_, _nt);
802  lhs_memb(z.no_cap_memb_, _nt);
803 
804  for (i = 0; i < z.no_cap_count_; ++i) {// parent axial current
805  Node* nd = z.no_cap_node_[i];
806  // following from global v_parent
807  NODERHS(nd) += NODED(nd) * NODEV(nd);
808  Node* pnd = _nt->_v_parent[nd->v_node_index];
809  if (pnd) {
810  NODERHS(nd) -= NODEB(nd) * NODEV(pnd);
811  NODED(nd) -= NODEB(nd);
812  }
813  }
814 
815  for (i = 0; i < z.no_cap_child_count_; ++i) {// child axial current
816  Node* nd = z.no_cap_child_[i];
817  // following from global v_parent

820  NODED(pnd) -= NODEA(nd);
